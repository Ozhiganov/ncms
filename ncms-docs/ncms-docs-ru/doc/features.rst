.. _features:

Возможности ηCMS
================

* Возможность одновременной работы множества сайтов на одном экземпляре ηCMS
* Возможность одновременной работы сотен редакторов контента сайтов
* Гибкое управления правами доступа к страницам сайтов
* Центральный репозиторий медиа-контента разделяемый сайтами в ηCMS
* Возможность изменения любой части сайта/страницы через административный интерфейс,
  включая статические ресурсы.
* ηCMS позволяет :ref:`расширять <extending>` функционал административной зоны сайта в соответствии
  с требованиями проекта
* Поддержка виртуальных хостов
* :ref:`Инструменты A/B тестирования <mtt>`
